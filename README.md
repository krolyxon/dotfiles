## prerequisites
```bash
sudo pacman -S git gum
```

## Usage
```bash
cd $HOME
git clone https://github.com/krolyxon/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
# Installation with no confirmation prompts
./install.sh --yes
```
