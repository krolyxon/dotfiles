## prerequisites
1. Git
2. Gum (For pretty prompts)
3. Base arch install

## Usage
```bash
cd $HOME
git clone https://github.com/krolyxon/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```
